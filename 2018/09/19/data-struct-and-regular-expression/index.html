The structure of the stack is like a cartridge magazine which is LIFO( Last In First Out ).

Here are some data struct code about normal stack.

// This is the data struct.
typedef struct
{
SElemType data[MAXSIZE];
int top;
}SqStack;

// And this is the stack data push function.
Status Push(SqStack *S, SElemType e)
{
if (S -> top == MAXSIZE - 1)
{
return ERROR;
}
S -> top++;
S -> data[S -> top] = e; 

return OK;

}

// And this is the stack data pop function.
Status Pop(SqStack S, SElemType e)
{
if(S -> top == 1)
return ERROR;
*e = S -> data[S -> top];
S -> top–;
return OK;
}

Then, here are some Elementary regular expressions. You can use it in shell commands or douwiki.

[:alnum:]alnum means alpha( English upper and lower-case letter ) num means number
[:alpha:]alpha means English upper and lower-case letter
[:blank:]blank means “space” or “tab”
[:cntrl:]cntrl means control buttons in keyboard like CR, LF, TAB, DEL
[:digit:]digit means number
[:graph:]graph means anything other rather than a blank character
[:lower:]lower means lower-case letter
[:print:]print means any character that can be printed out
[:punct:]punct means punctuation
[:upper:]upper means upper letter
[:space:]space means any character that creates a blank space like space, tab, CR and so on
[:xdigit:]xdigit means hex number
